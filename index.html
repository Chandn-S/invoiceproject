Frequently asked questions

What can I do with Swipe? ▼
With Swipe you can create all categories of Billing documents like Sales Invoices, Sales Returns, Purchase Invoices, Purchase Orders, Estimates, Manage Inventory in real time with Stock In & Stock Out options, and generate Delivery Challans for Invoices directly. Barcode Scanner and Thermal Print options are also available.

Is my data safe on Swipe? ▼
Swipe stores user data on its cloud using "Bank Grade" 256-bit SSL encryption. Data is not shared with any external parties. Please find the Privacy Policy and Terms for more details.

Can I add my Company's details and logo on Swipe? ▼
Yes, you can add your Company Name, GSTIN, Address and Logo to your invoices.

Can I use Swipe for Free? ▼
Swipe offers a free version with basic features. Yes, Swipe Mobile app & Web version are absolutely free to use for life-time to create unlimited invoices, purchases, quotations, manage customers/vendors, stock, payments, expenses, share bills via WhatsApp, Email or SMS etc. Start using Swipe for Free!

Who all can use Swipe? ▼
Swipe is a software for all kinds of small businesses, freelancers and Indian Startups. Anyone can start using Swipe to generate Invoices and other documents. All you need to do is sign-up with your mobile number and start using it!

Can I share my invoices or other documents with my customers? ▼
Yes, you can share your Invoices either on Whatsapp, Email or as an invoice link.

Is there any limit on the number of GSTINs I can add on Swipe? ▼
As long as the user remains the same, there is no limit to the number of GSTINs you can add on Swipe.

Is it possible to add multiple users? ▼
Yes

Can I control user permission for multiple users ? ▼
Yes with Swipe you can set user Role Settings for different users.

Does my data on Swipe web sync to Swipe mobile app? ▼
Yes, if you have used the same mobile number to login to Swipe on both mobile app as well as the web. All your data including invoices created, customers added, items added etc. will be synced.

Can I receive payments on Swipe? ▼
Yes

Can I send reminders for pending payments on Swipe? ▼
Yes , you can directly send reminders to customers for their outstanding payment and keep track of payment ledger on a single Dashboard Screen .


Can I recover my data if my system crashes? ▼
Yes, you need not worry about data recovery at all since all the data is stored on cloud servers. Even if your system or mobile crashes, you can login via another system with your mobile number and recover your data there.
